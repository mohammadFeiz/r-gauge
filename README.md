# r-gauger

### Install
```npm install r-gauger```
### Usage
ES6: <br>
```import RGauger from 'r-gauger;'``` <br>
ES5: <br>
```var RGauger = require('r-gauger');```
### Step By Step Tutorial

#### Step1
```<RGauger />```
#### Step2
add ranges props
``` 
... 
  ranges={[
     {value:33,color:'red'},
     {value:66,color:'orange'},
     {value:100,color:'yellow'},
  ]}
...
```
